Detailed Description

HPCG YAML_Element class, from the HPCG YAML_Element class for registering key-value pairs of performance data.

HPCG generates a collection of performance data for each run of the executable. YAML_Element, and the related YAML_Doc class, provide a uniform facility for gathering and reporting this data using the YAML text format.
