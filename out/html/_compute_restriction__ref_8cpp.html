Routine to compute the coarse residual vector.

Parameters
[in,out]A- Sparse matrix object containing pointers to mgData->Axf, the fine grid matrix-vector product and mgData->rc the coarse residual vector.
[in]rf- Fine grid RHS.

Note that the fine grid residual is never explicitly constructed. We only compute it for the fine grid points that will be injected into corresponding coarse grid points.

Returns
Returns zero on success and a non-zero value otherwise.
