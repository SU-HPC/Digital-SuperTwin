31 <h3> HPCG Benchmark Release 2.1</h3>
32 
33 <p>HPCG 2.0/2.1 provides the first implementation of a multigrid preconditioner for the conjugate gradient method.
34 <ul>
35 <li> HPCG 2.0/2.1replaces the additive Schwarz preconditioner with a synthetic multigrid approach. </li>
36 <li> The number of coarse grid levels is parametrized but fixed for production benchmark runs (presently set to 3 levels of coarsening).</li>
37 <li> We use injection as the grid transfer operator and symmetric Gauss-Seidel as the pre and post smoother.</li>
38 <li> All kernels from HPCG 1.1 remain important and no new kernels were added.
39 <li> The biggest impact of the multigrid preconditioner is that all computation and communication kernels will execute on
40  a nested sequence of coarse grids where each grid level is 8 times smaller than the previous level.</li>
41 <li> Every attempt was made to assure that the kernels are not unnecessarily different between HPCG 1.1 and HPCG 2.0.</li>
42 <li> Even so, some changes were necessary to supported the nested hierarchy.</li>
43 <li> We did not implement a true multigrid preconditioner. In particular, we do not have a "bottom" solver for the coarsest grid.</li>
44 <li> We also added a Vector struct to facility easier implementation of kernels on discrete devices such as GPUs.</li>
45 </p>
