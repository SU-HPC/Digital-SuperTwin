Computes one step of symmetric Gauss-Seidel:

Assumption about the structure of matrix A:

  • Each row 'i' of the matrix has nonzero diagonal value whose address is matrixDiagonal[i]
  • Entries in row 'i' are ordered such that:
    • lower triangular terms are stored before the diagonal element.
    • upper triangular terms are stored after the diagonal element.
    • No other assumptions are made about entry ordering.

Symmetric Gauss-Seidel notes:

  • We use the input vector x as the RHS and start with an initial guess for y of all zeros.
  • We perform one forward sweep. x should be initially zero on the first GS sweep, but we do not attempt to exploit this fact.
  • We then perform one back sweep.
  • For simplicity we include the diagonal contribution in the for-j loop, then correct the sum after
Parameters
[in]Athe known system matrix
[in]rthe input vector
[in,out]xOn entry, x should contain relevant values, on exit x contains the result of one symmetric GS sweep with r as the RHS.
Warning
Early versions of this kernel (Version 1.1 and earlier) had the r and x arguments in reverse order, and out of sync with other kernels.
Returns
returns 0 upon success and non-zero otherwise
