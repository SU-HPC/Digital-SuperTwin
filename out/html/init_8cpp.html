◆ HPCG_Init()

int HPCG_Init ( int *  argc_p,
char ***  argv_p,
HPCG_Params &  params 
)

Initializes an HPCG run by obtaining problem parameters (from a file or command line) and then broadcasts them to all nodes. It also initializes login I/O streams that are used throughout the HPCG run. Only MPI rank 0 performs I/O operations.

The function assumes that MPI has already been initialized for MPI runs.

Parameters
[in]argc_pthe pointer to the "argc" parameter passed to the main() function
[in]argv_pthe pointer to the "argv" parameter passed to the main() function
[out]paramsthe reference to the data structures that is filled the basic parameters of the run
Returns
returns 0 upon success and non-zero otherwise
See also
