Routine to compute the dot product of two vectors.

This routine calls the reference dot-product implementation by default, but can be replaced by a custom routine that is optimized and better suited for the target system.

Parameters
[in]nthe number of vector elements (on this processor)
[in]x,ythe input vectors
[out]resulta pointer to scalar value, on exit will contain the result.
[out]time_allreducethe time it took to perform the communication between processes
[out]isOptimizedshould be set to false if this routine uses the reference implementation (is not optimized); otherwise leave it unchanged
Returns
returns 0 upon success and non-zero otherwise
