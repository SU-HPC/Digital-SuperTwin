Detailed Description

The OutputFile class for the uniform collecting and reporting of performance data for HPCG.

The OutputFile class facilitates easy collecting and reporting of key-value-formatted data that can be then registered with the HPCG results collection website. The keys may have hierarchy key1::key2::key3=val with double colon :: as a separator. A sample output may look like this (note how "major" and "micro" keys repeat with different ancestor keys):

version=3.2.1alpha
version::major=3
version::minor=2
version::micro=1
version::release=alpha
axis=xyz
axis::major=x
axis::minor=y
