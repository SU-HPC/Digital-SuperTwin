Reference routine to compute an approximate solution to Ax = b

Parameters
[in,out]AThe known system matrix
[in,out]dataThe data structure with all necessary CG vectors preallocated
[in]bThe known right hand side vector
[in,out]xOn entry: the initial guess; on exit: the new approximate solution
[in]max_iterThe maximum number of iterations to perform, even if tolerance is not met.
[in]toleranceThe stopping criterion to assert convergence: if norm of residual is <= to tolerance.
[out]nitersThe number of iterations actually performed.
[out]normrThe 2-norm of the residual vector after the last iteration.
[out]normr0The 2-norm of the residual vector before the first iteration.
[out]timesThe 7-element vector of the timing information accumulated during all of the iterations.
[in]doPreconditioningThe flag to indicate whether the preconditioner should be invoked at each iteration.
Returns
Returns zero on success and a non-zero value otherwise.
