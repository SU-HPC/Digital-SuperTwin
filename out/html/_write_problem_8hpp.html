Routine to dump:

  • matrix in row, col, val format for analysis with MATLAB
  • x, xexact, b as simple arrays of numbers.

Writes to A.dat, x.dat, xexact.dat and b.dat, respectivly.

NOTE: THIS CODE ONLY WORKS ON SINGLE PROCESSOR RUNS

Read into MATLAB using: 

load A.dat
A=spconvert(A);
load x.dat
load xexact.dat
load b.dat
Parameters
[in]geomThe description of the problem's geometry.
[in]AThe known system matrix
[in]bThe known right hand side vector
[in]xThe solution vector computed by CG iteration
[in]xexactGenerated exact solution
Returns
Returns with -1 if used with more than one MPI process. Returns with 0 otherwise.
