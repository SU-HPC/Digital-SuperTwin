void GenerateCoarseProblem ( const SparseMatrix &  Af)

Routine to construct a prolongation/restriction operator for a given fine grid matrix solution (as computed by a direct solver).

Parameters
[in,out]Af- The known system matrix, on output its coarse operator, fine-to-coarse operator and auxiliary vectors will be defined.

Note that the matrix Af is considered const because the attributes we are modifying are declared as mutable.
