Test the correctness of the Preconditined CG implementation by using a system matrix with a dominant diagonal.

Parameters
[in]geomThe description of the problem's geometry.
[in]AThe known system matrix
[in]datathe data structure with all necessary CG vectors preallocated
[in]bThe known right hand side vector
[in,out]xOn entry: the initial guess; on exit: the new approximate solution
[out]testcg_datathe data structure with the results of the test including pass/fail information
Returns
Returns zero on success and a non-zero value otherwise.
