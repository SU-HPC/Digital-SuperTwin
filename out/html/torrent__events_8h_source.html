6 /* PRELIMINARY EVENT ENCODING
7  * 0x0000_0000 - 0x00FF_FFFF = PowerPC core events
8  * 0x0100_0000 - 0x01FF_FFFF = Torrent events
9  * 0x0200_0000 - 0xFFFF_FFFF = reserved
10  * For Torrent events:
11  * Reserve encodings 0x0..0x00FF_FFFF for core PowerPC events.
12  * For Torrent events
13  * 0x00F0_0000 = Torrent PMU id
14  * 0x000F_0000 = PMU unit number (e.g. 0 for MCD0, 1 for MCD1)
15  * 0x0000_FF00 = virtual counter number (unused on MCD)
16  * 0x0000_00FF = PMC mux value (unused on Util, MMU, CAU)
17  * (Note that some of these fields are wider than necessary)
18  *
19  * The upper bits 0xFFFF_FFFF_0000_0000 are reserved for attribute
20  * fields.
21  */
22
